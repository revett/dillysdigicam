# Dilly's Digicam

Drop your photos, pick your favs, and Dilly will help give them that 00s digicam aesthetic.

## Local Development

```sh
bun install
bun run dev
```

Opens at `http://localhost:1234`.

## Production Build

```sh
bun run build
```

Outputs to `dist/`.

## Linting

```sh
bun run lint
bun run lint:fix
```
