# AGENTS.md

## Rules

1. Must discuss adding a new library with the team before adding it
1. Simple always wins
1. Make everything stupidly fast for end users
1. Always leave the codebase in a better state than you found it, including documentation!
1. Ensure to lint/format code after making changes

## Choices

- We use Bun
- We use Tailwind for styling
- We use Parcel for bundling
- We use Biome for linting
- We use GitHub Actions for CI
- We use GitHub Pages for deployment
- We use AlpineJS for our framework
